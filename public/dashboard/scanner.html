Mac Scanner

Download and run the scanner on macOS. Python 3 required.

‚¨áÔ∏è Download for Mac

üìã Mac Instructions

  1. Download the ZIP file above
  2. Extract to a folder
  3. Open Terminal (Applications ‚Üí Utilities ‚Üí Terminal)
  4. Navigate to folder: cd ~/Downloads/truthvault-scanner
  5. Make executable: chmod +x run_scanner.sh
  6. Run: ./run_scanner.sh
  7. Enter your email and auth token when prompted

No Python? Install with Homebrew:

Linux Scanner

Download and run the scanner on Ubuntu, Debian, Fedora, etc.

‚¨áÔ∏è Download for Linux

üìã Linux Instructions

  1. Download the ZIP file above
  2. Extract: unzip truthvault-scanner-linux.zip
  3. cd truthvault-scanner
  4. chmod +x run_scanner.sh
  5. ./run_scanner.sh
  6. For better device detection, run with sudo: sudo ./run_scanner.sh

Install Python if needed:
Ubuntu/Debian: sudo apt install python3 python3-pip
Fedora: sudo dnf install python3 python3-pip
