## ğŸ› IF SOMETHING DOESN'T WORK

### Common Issues:

**"Database not found" error:**
- Make sure you ran setup-databases.php first
- Check that databases/ folder exists
- Check file permissions

**"Token invalid" error:**
- Logout and login again
- Check JWT secret in config.php
- Check token expiration time

**"Cannot generate keys" error:**
- Make sure TweetNaCl.js is loading
- Check browser console for errors
- Try different browser

**"Server not found" error:**
- Run admin/update-servers.php first
- Check servers.db has 4 servers
- Check server status in admin panel

**PayPal not working:**
- Verify PayPal credentials in main.db settings table
- Check webhook URL in PayPal dashboard
- Check api/billing/webhook.php exists
