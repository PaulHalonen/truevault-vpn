Get TrueVault VPN running on Windows or Mac

1

Download WireGuard

Download the official WireGuard app for Windows:

üì• Download for Windows

Run the installer and follow the prompts. No configuration needed during install.

Download WireGuard from the Mac App Store:

üì• Get on Mac App Store
2

Get Your Configuration

  1. Go to your TrueVault dashboard
  2. Click "+ Add Device"
  3. Name it (e.g., "My Laptop")
  4. Choose your server location
  5. Click "Download Config"

Save the .conf file somewhere you can find it (Desktop or Downloads).

3

Import Configuration

  1. Open WireGuard from Start menu
  2. Click "Import tunnel(s) from file"
  3. Select your downloaded .conf file
  4. The tunnel appears in your list
  1. Open WireGuard from Applications
  2. Click "Import tunnel(s) from file" or drag the file onto the window
  3. Allow the system extension when prompted
  4. The tunnel appears in your list
4

Connect

  1. Select your TrueVault tunnel
  2. Click "Activate"
  3. Status changes to "Active" with green indicator
