Get Your VPN Configuration

Go to your TrueVault dashboard and add a new device:

  1. Open your dashboard
  2. Click "+ Add Device"
  3. Name it (e.g., "My Android Phone")
  4. Choose your preferred server location
  5. Click "Generate Config"

You'll see a QR code and a Download button.

3

Import Configuration

üì∑ Using QR Code:

  1. Open WireGuard app on your phone
  2. Tap the blue + button
  3. Select "Scan from QR code"
  4. Point your camera at the QR code on your computer screen
  5. Name the tunnel (e.g., "TrueVault")
  6. Tap "Create Tunnel"

üìÅ Using Config File:

  1. Download the .conf file to your phone
  2. Open WireGuard app
  3. Tap the blue + button
  4. Select "Import from file or archive"
  5. Navigate to Downloads and select your file
  6. Tap "Create Tunnel"

üîß Fix: Can't Find Config File?

If WireGuard shows "No files found" when importing, Android likely saved it with the wrong extension.

The Problem

Android sometimes saves truevault.conf as truevault.conf.txt

Quick Fix Options:

Option A: Use a File Manager
  1. Install Files by Google (free)
  2. Go to Downloads folder
  3. Find your config file
  4. Long-press ‚Üí Rename
  5. Remove .txt from the end
  6. Try importing again in WireGuard
Option B: Use QR Code Instead

Go back to your dashboard and scan the QR code - it's faster and avoids this issue entirely!

üí° Pro Tip

Always use the QR code method on Android - it's faster and never has file extension issues!
