## How to Switch Servers

Want to switch from New York to Dallas for Netflix? Easy!

### On Your Dashboard:

1. Find your device card
2. Click **"üîÑ Switch Server"**
3. Select the new server (e.g., Dallas)
4. Click **"Switch"** and wait 3 seconds
5. Download the new config file
6. **Remove the old tunnel** in WireGuard app
7. **Import the new config** file
8. Toggle connection ON
9. ‚úÖ You're now on the new server!

## Frequently Asked Questions

### Q: Can I use the same config on multiple devices?
**A:** Yes! You can download the same config multiple times and use it on different devices. However, each device counts toward your device limit.

### Q: What happens if I switch servers?
**A:** You get a new config file with a new IP address. Your old config stops working. Just import the new config file in WireGuard.

### Q: Can I switch back to my old server?
**A:** Yes! Just click "Switch Server" again and select your old server. You'll get a fresh config file.

### Q: Do I need to generate new keys each time?
**A:** No! We automatically reuse your existing keys when you switch servers or add new devices.

### Q: What if I hit my device limit?
**A:** You can remove an unused device to free up a slot, or upgrade your plan for more devices.

### Q: Can I rename my device?
**A:** Yes! Just click the edit icon on your device card in the dashboard.

## Troubleshooting

### "Device limit reached"
- **Solution:** Remove an old device or upgrade your plan

### "Server is offline"
- **Solution:** Try a different server or wait a few minutes

### "VPN not connecting"
- **Solution:** 
  1. Check if you imported the correct config file
  2. Make sure you toggled the connection ON
  3. Try switching to a different server
  4. Contact support if issue persists

### "Connection drops frequently"
- **Solution:** 
  1. Try New York server (most stable)
  2. Check your internet connection
  3. Make sure WireGuard is allowed through your firewall
